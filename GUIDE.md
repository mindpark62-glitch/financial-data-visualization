# 📖 사용 가이드

## 🎯 시작하기

### 1. corp.xml 파일 준비

현재 프로젝트에는 테스트용 샘플 데이터가 포함되어 있습니다. 실제 전체 회사 데이터를 사용하려면:

1. [OpenDart 공시정보활용](https://opendart.fss.or.kr/disclosureinfo/fnltt/dwld/main.do)에서 `CORPCODE.zip` 다운로드
2. 압축 해제 후 `CORPCODE.xml` 파일을 `public/corp.xml`로 복사
3. 개발 서버 재시작

```bash
# Windows PowerShell
Copy-Item "다운로드경로\CORPCODE.xml" "public\corp.xml" -Force

# 개발 서버 재시작
npm run dev
```

### 2. 애플리케이션 실행

```bash
# 개발 서버 시작
npm run dev

# 브라우저에서 http://localhost:5173 접속
```

## 💡 사용 방법

### Step 1: 회사 검색

1. 검색창에 회사명 또는 종목코드 입력
   - 예: "삼성전자", "005930", "Samsung"
2. 자동완성 목록에서 원하는 회사 선택
3. 선택된 회사 정보 확인

**검색 팁:**
- 최소 2글자 이상 입력해야 검색됩니다
- 한글, 영문, 숫자 모두 검색 가능
- 최대 20개 결과까지 표시됩니다

### Step 2: 재무 데이터 조회

1. **사업연도 선택**
   - 2015년 이후 데이터 제공
   - 최근 10년 선택 가능

2. **보고서 유형 선택**
   - 사업보고서: 연간 재무제표 (가장 상세)
   - 반기보고서: 상반기 재무제표
   - 1분기보고서: 1분기 재무제표
   - 3분기보고서: 3분기 재무제표

3. "재무 데이터 조회" 버튼 클릭

**조회 팁:**
- 사업보고서는 전전기 데이터까지 포함
- 분기/반기보고서는 당기와 전기만 비교

### Step 3: 데이터 시각화 확인

조회가 완료되면 다음 정보가 표시됩니다:

#### 📈 차트
- **막대 차트**: 주요 재무 지표를 시각적으로 비교
- **색상 구분**:
  - 회색: 전전기
  - 파란색: 전기
  - 초록색: 당기
- **인터랙티브**: 마우스를 올리면 상세 금액 표시

#### 📋 테이블
- **주요 계정**: 자산, 부채, 자본, 매출, 이익 등
- **증감률**: 전기 대비 증감률 자동 계산
- **색상 표시**:
  - 초록색: 증가
  - 빨간색: 감소

#### 재무제표 구분
- **연결재무제표 (CFS)**: 자회사 포함 전체 그룹 재무 상태
- **개별재무제표 (OFS)**: 해당 회사만의 재무 상태

### Step 4: AI 재무 분석

1. "AI 분석 시작하기" 버튼 클릭
2. AI가 재무 데이터를 분석하는 동안 대기 (약 5-10초)
3. 분석 결과 확인

**AI 분석 내용:**
- 📊 전반적인 재무 상태 요약
- 📈 주요 재무 지표 변화 분석
- ✅ 긍정적인 측면
- ⚠️ 주의해야 할 점
- 💡 종합 평가

**분석 팁:**
- 전문 용어를 최소화한 쉬운 설명
- 구체적인 숫자와 비율 제공
- "다시 분석하기"로 재분석 가능

## 🔧 문제 해결

### 회사 검색이 안 돼요
- corp.xml 파일이 public 폴더에 있는지 확인
- 개발 서버를 재시작해보세요
- 브라우저 콘솔(F12)에서 에러 확인

### 재무 데이터 조회 실패
- OpenDart API 키가 올바른지 확인
- 해당 연도/보고서 데이터가 존재하는지 확인
- API 일일 요청 제한(20,000건) 확인

**에러 메시지별 해결:**
- "013: 조회된 데이터가 없습니다"
  → 해당 연도/보고서가 없음. 다른 연도 시도
- "020: 요청 제한을 초과하였습니다"
  → 다음 날 다시 시도
- "010: 등록되지 않은 키입니다"
  → API 키 재확인

### AI 분석이 실패해요
- Gemini API 키가 올바른지 확인
- 인터넷 연결 확인
- API 무료 티어 제한 확인
- 잠시 후 다시 시도

### 차트가 제대로 표시되지 않아요
- 브라우저를 최신 버전으로 업데이트
- 브라우저 캐시 삭제 후 새로고침
- 다른 브라우저에서 시도

## 📊 재무 지표 설명

### 재무상태표 (BS)
- **자산총계**: 회사가 보유한 모든 자산
- **부채총계**: 회사가 갚아야 할 빚
- **자본총계**: 자산에서 부채를 뺀 순수 자본

### 손익계산서 (IS)
- **매출액**: 제품/서비스 판매로 벌어들인 돈
- **영업이익**: 본업으로 벌어들인 이익
- **당기순이익**: 세금 등을 제외한 최종 이익

### 재무 건전성 판단
- 자산 > 부채: 건전한 상태
- 영업이익 증가: 본업이 잘되고 있음
- 당기순이익 증가: 전반적으로 수익성 개선
