일부 회사는 특정 연도의 보고서가 없을 수 있습니다.
- 다른 연도 선택
- 다른 보고서 유형 선택 (반기, 분기)

- 2023년 이후 데이터는 회사별로 다름

## 🔄 전체 회사 데이터 사용하기

현재는 주요 60개 회사만 포함되어 있습니다.

### 전체 회사 데이터 (약 8,000개) 사용하려면:

1. [OpenDart 공시정보](https://opendart.fss.or.kr/disclosureinfo/fnltt/dwld/main.do) 접속
2. **CORPCODE.zip** 다운로드
3. 압축 해제 후 `CORPCODE.xml` 파일 확인
4. `public/corp.xml`로 복사
5. 서버 재시작

```bash
# Windows PowerShell
Copy-Item "다운로드경로\CORPCODE.xml" "public\corp.xml" -Force
npm run dev
```
