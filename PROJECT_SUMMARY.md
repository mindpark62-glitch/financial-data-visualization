# 📊 프로젝트 요약

## 프로젝트 개요

**재무 데이터 시각화 분석 서비스**는 누구나 쉽게 기업의 재무 정보를 이해할 수 있도록 돕는 웹 애플리케이션입니다.

## ✨ 핵심 기능

### 1. 회사 검색 시스템
- **기술**: XML 파싱 (xml2js), 실시간 검색
- **특징**:
  - corp.xml 데이터베이스 활용
  - 자동완성 기능
  - 디바운스 최적화 (300ms)
  - 메모리 캐싱으로 빠른 재검색

### 2. 재무 데이터 조회
- **API**: OpenDart (금융감독원 전자공시시스템)
- **지원 보고서**:
  - 사업보고서 (연간)
  - 반기보고서
  - 1분기/3분기보고서
- **데이터**:
  - 연결재무제표 (CFS)
  - 개별재무제표 (OFS)
  - 2015년 이후 데이터

### 3. 데이터 시각화
- **차트 라이브러리**: Recharts
- **시각화 요소**:
  - 인터랙티브 막대 차트
  - 당기/전기/전전기 비교
  - 커스텀 툴팁
  - 반응형 디자인
- **주요 지표**:
  - 자산총계, 부채총계, 자본총계
  - 매출액, 영업이익, 당기순이익

### 4. AI 재무 분석
- **AI 모델**: Google Gemini 2.0 Flash
- **분석 내용**:
  - 재무 상태 요약
  - 주요 지표 변화 분석
  - 긍정적 측면 및 주의사항
  - 종합 평가
- **특징**: 일반인도 이해하기 쉬운 설명
